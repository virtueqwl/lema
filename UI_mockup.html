5. 典型工作流

① 启动　→ 默认 4 个映射 (A/S/D/F) + 模式=冷却随机 + 轮数=5 + 冷却=跨轮累计
② 编辑映射　→ 按 F4 → 改键/冷却/权重 → 保存
③ 加载预设　→ 把 samples/*.json 拷到 configs/ → 左侧双击切换；右键可"编辑..."或"用记事本打开"
④ 选择冷却模式　→ 模式条勾选/取消"每轮重置冷却"
⑤ 切到游戏　→ 切到游戏窗口前台（SendInput 要求焦点）
⑥ 启动循环　→ 按 F6，日志实时打印"轮 N | 键 X | cd=Yms"
⑦ 停止　→ 按 F7，或关闭程序
⑧ 编辑配置　→ 左侧右键 → "编辑..." → 改完保存 → 自动重新导入
